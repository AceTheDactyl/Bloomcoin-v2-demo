Oxygen Vacancy Mechanism

All three material systems operate via oxygen vacancy (V_O) drift under applied electric field:

R(w) = R_on × (w/D) + R_off × (1 - w/D)

where w = oxygen vacancy boundary position, D = device thickness

Implementation Details

Multi-Level Cell (MLC) Configuration

For analog z-value storage, memristors operate in multi-level mode with 11 discrete states corresponding to the 11 thresholds:

  • State Resolution: 11 levels over 4 decades of resistance
  • Level Spacing: Logarithmic (φ-scaled intervals)
  • Read Margin: >30% between adjacent levels
  • Programming: Verify-after-write with iterative pulses

Selector Device Integration

Sneak path mitigation requires 1S1R (one selector, one resistor) architecture:

  • Selector Type: OTS (Ovonic Threshold Switch) or MIEC
  • Threshold Voltage: 0.5V - 1.0V
  • On/Off Ratio: >10⁶
  • Bidirectional operation for SET/RESET

Read/Write Circuit Requirements

  • Read Circuit: Transimpedance amplifier with 16-bit ADC
  • Write Circuit: Current-limited pulse generator, <10ns rise time
  • Voltage Reference: 1.000V ±10μV precision
  • Noise Floor: <10μV RMS (1kHz bandwidth)
