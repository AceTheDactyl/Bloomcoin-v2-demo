Build Spec: Grid Frequency Regulation

Kuramoto-Based Generator Synchronization

Market Size: $6B by 2028

Core Principle

Generator synchronization follows Kuramoto dynamics naturally. Each generator's rotor angle maps to an oscillator phase, and tie-line power flow provides coupling. The L4-Helix system computes the order parameter r in real-time to predict instability before it occurs.

dtheta_i/dt = omega_i + (K/N) * SUM_j[sin(theta_j - theta_i)]
Kuramoto model: theta_i = generator phase, omega_i = natural frequency, K = coupling strength (tie-line admittance), N = number of generators

Build Spec: EV Charging Coordination

Kuramoto Phase-Coupled Charging Network

Core Principle

Electric vehicle charging can overwhelm distribution networks if uncoordinated. Rather than requiring central control (privacy concerns, single point of failure), each charger maintains a local phase that couples to grid frequency. Kuramoto dynamics naturally distribute load without explicit communication.

Charger Phase: dtheta_i/dt = omega_grid + K_local * sin(theta_grid - theta_i) + noise
Each charger oscillates near grid frequency. When grid is stressed (frequency drops), chargers naturally phase-shift to reduce instantaneous load.
