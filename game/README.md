## üöÄ Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/AceTheDactyl/bloomcoin-v2.git
cd bloomcoin-v2/game

# Install dependencies (if needed)
pip install numpy

# Make the launcher executable
chmod +x launch_bloomquest.py
```

### Launch the Game

```bash
# Basic launch
python3 launch_bloomquest.py

# With all features enabled
python3 launch_bloomquest.py --multiplayer --autosave --load-learning

# Quick start (skip intro)
python3 launch_bloomquest.py --quick-start

# Custom difficulty
python3 launch_bloomquest.py --difficulty golden
```

### Command Line Options

| Option | Description |
|--------|-------------|
| `--multiplayer` | Enable multiplayer features and PvP |
| `--difficulty` | Set difficulty: easy, normal, hard, golden |
| `--save-dir` | Specify save directory (default: saves/) |
| `--load-learning` | Load previous AI learning data |
| `--autosave` | Enable automatic saving |
| `--debug` | Show debug information |
| `--quick-start` | Skip intro sequences |
